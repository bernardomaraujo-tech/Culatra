# Roadmap — Pesca Culatra

## Fase 1 — MVP estático

- [x] Home com 3 zonas.
- [x] Separadores: Meteorologia, Marés, Lua e Pesca.
- [x] Score por zona.
- [x] Consumo Open-Meteo Weather.
- [x] Consumo Open-Meteo Marine.
- [x] Fallback para marés.
- [x] Publicação simples em GitHub Pages.

## Fase 2 — Dados reais e validação

- [ ] Validar coordenadas exatas de Sul, Norte e Ponta.
- [ ] Ligar marés reais via API.
- [ ] Adicionar validação IPMA.
- [ ] Adicionar link para marés oficiais do Instituto Hidrográfico.
- [ ] Adicionar avisos legais DGRM e ICNF.

## Fase 3 — Histórico e personalização

- [ ] Criar registo manual de pescarias.
- [ ] Guardar espécie, hora, zona, isco e resultado.
- [ ] Ajustar score com base em histórico real.
- [ ] Criar recomendação por espécie.
- [ ] Criar alertas automáticos para boas janelas.

## Fase 4 — Produto

- [ ] PWA instalável.
- [ ] Modo offline parcial.
- [ ] Backend leve com Supabase.
- [ ] Dashboard de histórico.
- [ ] Página pública de previsão diária.
